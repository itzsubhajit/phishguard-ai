# 🛡️ PhishGuard AI

An AI-powered phishing detection system that identifies malicious URLs, scam messages, and suspicious content using machine learning techniques.

---

## 🚀 Features

- 🔍 Detects phishing URLs in real-time  
- 🤖 Machine Learning-based classification  
- 📊 Dataset generation and preprocessing  
- 🌐 Web-based interface for user interaction  
- ⚡ Lightweight and cloud-deployable  

---

## 🧠 How It Works

1. User inputs a URL or message  
2. System extracts important features  
3. Machine learning model analyzes patterns  
4. Generates a phishing probability score  
5. Displays result with safety suggestions

### 2️⃣ Install Dependencies
